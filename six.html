Html tags:

  1. html

    2. html tag is used to define an html document, it represents the root element of the webpage

  2. head

    3. head contains information which are not visible on webpage, it contains information like metadata of the webpage, title, seo data's

  3. title

    4. title tag is used to define the title of the webpage, it is displayed on the tab of the browser

  4. meta

    5. meta tag is used to define metadata of the webpage, it is not visible on the webpage, it contains information like author, description, keywords, viewport, charset, etc

  5. body

    6. body tag is used to define the body of the webpage, it contains all the visible elements of the webpage

  6. form

    7. form tag is used to define a form, it contains all the form elements like input, select, textarea, button, etc

  7. a

    8. a tag is used to define a hyperlink, it is used to link a webpage to another webpage

  8. h1

    9. h1 tag is used to define a heading, it is the largest heading tag

  9. strong

    10. strong tag is used to define a text with strong importance, it is displayed in bold

  10. table

    11. table tag is used to define a table, it contains all the table elements like tr, td, th, etc
